# 蚊子探测器

## 开发状态
**当前版本: 1.0**

- 正在进行基础功能开发。

---

这是一款安卓应用，旨在使用设备的摄像头和实时图像处理技术，帮助用户在房间内定位静止的蚊子。

## 功能特性

- **实时相机画面**: 显示来自设备相机的实时视频流。
- **自定义图像滤镜**: 对实时画面进行处理，为背景应用一层带蓝色色调的灰度滤镜。
- **目标高亮**: 应用会分析视频流，以寻找可能是蚊子的小、暗、静止的物体。
    - 潜在目标会以从**黄色**（低置信度）到**红色**（高置信度）的渐变色进行高亮显示。
- **点击对焦**: 触摸预览画面的任意位置，可以快速对该点进行自动对焦和自动测光。
- **手势变焦**: 支持通过双指捏合手势来平滑地放大和缩小相机画面。
- **闪光灯控制**: 屏幕上提供一个图标按钮，用于打开和关闭设备的闪光灯（手电筒模式）。

## 工作原理

本应用使用 `CameraX` 进行相机相关操作。来自相机的每一帧画面都会被实时分析：

1.  图像首先被转换为带蓝色色调的灰度位图，作为背景。
2.  一个独立的分析过程会识别出比预设阈值更暗的像素点。
3.  这些暗色像素会根据其黑暗程度，被着上从黄色到红色的颜色。
4.  然后，这个高亮图层会被合成到蓝色背景图层之上。
5.  最终合成的图像会经过旋转校正，以匹配设备的屏幕方向，然后显示在屏幕上。
